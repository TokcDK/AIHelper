Data - сюда игру \ game file here

Mods - папки с модами типа Mods\Mod1 , Mods\Mod2 и т.п., каждая папка мода считается как папка Data и еи файлы в ней будут загружаться в игру как если бы они были в папке Data.

MO - здесь будут файлы настроек Mod Organizer, профили с списком модов, категории и настройки самого Mod Organizer для этой игры. Файлы будут созданы автоматом при запуске AIHelper, если папка пустая.


Вместо папки с сборкой игры можно создать в папке Games тестовый файл, где первой строчкой будет путь к сборке, в случае, когда сборки на разных дисках или не хочется перемещать в Games.
